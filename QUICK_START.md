# Jesse+ 快速访问指南

## 🌐 访问前端系统

### 方式1: 直接访问
在浏览器中打开：
```
http://43.134.38.231:8060
```

### 方式2: SSH端口转发（推荐）
如果无法直接访问，使用SSH端口转发：
```bash
ssh -L 8060:localhost:8060 ubuntu@43.134.38.231
```
然后在浏览器访问：
```
http://localhost:8060
```

## 🚨 故障排查

### 无法访问前端
1. 检查服务状态：`pm2 list`
2. 查看日志：`pm2 logs jesse-web-v3`
3. 检查端口：`netstat -tlnp | grep 8060`
4. 检查防火墙：`sudo ufw status`

### 数据不更新
1. 检查交易所API配置
2. 查看交易系统日志：`pm2 logs jesse-trading-system`
3. 点击页面上的"🔄 刷新"按钮

### 策略进化未运行
1. 在侧边栏启用"策略自动进化"
2. 检查进化系统日志：`pm2 logs jesse-auto-evolution`

## 💡 使用技巧

1. **实时监控**：页面会自动更新数据，无需手动刷新
2. **多标签页**：可以同时打开多个标签页查看不同功能
3. **套利机会**：在"多交易所监控"页面关注价差超过0.1%的机会
4. **策略优化**：启用自动进化后，系统会持续优化交易策略
5. **风险控制**：定期查看风险监控页面，确保回撤在可控范围内
